Add desired class to body.

== Description ==

This plugin allows you to add a custom class to the body tag of your WordPress website. You can set a global class for the body or specify custom classes for individual posts, pages, and custom post types.

== Installation ==

1. Upload the `add-class-to-body` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' > 'Add Class to Body' to configure the plugin settings.
4. Set the global class for the body and choose post types where you want to hide the custom class option.
5. Edit individual posts/pages to add custom classes using the meta box provided.

== Frequently Asked Questions ==

= How do I add a custom class to the body? =

After activating the plugin, go to 'Settings' > 'Add Class to Body' to set a global class for the body. You can also add custom classes to individual posts/pages by editing them and using the meta box provided.
